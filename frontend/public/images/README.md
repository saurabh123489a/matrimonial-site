# Images Directory

Place your wedding ceremony images here:

## Required Images:

1. **bridal-lehenga.jpg** - Red & Gold Lehenga bridal photography image (landscape, 1920x1080 recommended)
   - Used in Hero section and CTA section

## Image Specifications:

- **Format**: JPG or PNG
- **Size**: Landscape orientation (1920x1080 or similar aspect ratio)
- **Quality**: High quality for best display
- **File Size**: Optimize for web (under 500KB recommended)

## How to Add Images:

1. Place your image files in this directory (`frontend/public/images/`)
2. Name them according to the references in `app/page.tsx`
3. The images will be automatically available at `/images/filename.jpg`
